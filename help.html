Menggambar Bentuk Baru

Menggambar Garis

1. Pilih "Line" pada menu shape

2. Pilih warna pada menu color

3. Pilih draw

4. Klik pada canvas untuk menentukan lokasi titik pertama garis

5. Klik pada canvas untuk menentukan lokasi titik kedua garis

Menggambar Kotak

1. Pilih "Square" pada menu shape

2. Pilih warna pada menu color

3. Pilih draw

4. ................

5. ................

Menggambar Polygon

1. Pilih "Polygon" pada menu shape

2. Pilih warna pada menu color

3. Pilih draw

4. Klik pada canvas sebanyak minimal 3 kali untuk menentukan lokasi titik-titik polygon

5. Klik pada canvas untuk menentukan lokasi titik-titik selanjutnya pada polygon

6. Setelah mendapat bentuk yang diinginkan, tekan "Enter" pada keyboard

Edit / Mengubah Bentuk yang Sudah Ada

1. Pilih "Edit" pada "radio button"

2. Pilih titik yang ingin diubah dari suatu garis, kotak, atau polygon

3. Klik pada canvas untuk menentukan lokasi titik yang baru dari bentuk yang dipilih

Mengubah Warna Polygon

1. Pilih "Edit" pada "radio button"

2. Pilih titik yang ingin diubah dari suatu titik pada polygon

3. Pilih warna baru pada menu color

4. Klik canvas pada posisi titik yang sama ketika memilih titik yang ingin diubah warnanya

Menyimpan Gambar

1. Setelah selesai menggambar, tekan tombol "Save"

2. File berisi konfigurasi gambar akan tersimpan dalam bentuk text (.txt)
